Tablas en HTML

Una tabla en HTML, es un conjunto de celdas donde podemos alojar contenidos
tales como: texto, imágenes, vídeos, audios, tablas, formularios, etc.
Antiguamente, era un elemento fundamental para la creación de páginas web.
Era un elemento que utilizábamos por lo tanto, para poder maquetar nuestras
páginas webs con el contenido que queríamos.
Con la llegada de HTML5, se empezó a utilizar etiquetas semánticas(aside, nav,
header, etc) y contenedores (divs), que nos permiten tener una mayor flexibilidad
y adaptar nuestras páginas webs de manera mucho más dinámica y visualmente mucho
más atractivas.
Las características o estructura principal que forma una tabla son los siguientes elementos:
  • table: con esta etiqueta, definimos que vamos a utilizar
    una tabla
  • tr: sirve para definir las filas de una tabla
  • td: sirve para definir las celdas de una tabla

Propiedades de las tablas

  • table: inicio y fin de una tabla
  • tr: para definir el número de filas de las tablas
  • td: para definir el número de celdas de las tablas
  • caption: es para poner un título encima o debajo de una tabla
  • alinear caption: Utilizamos el atributo align con top o bottom
  • thead: se utiliza para definir la cabecera de una tabla
  • th: Define una celda de tipo cabecera en la tabla
  • tbody: Define el cuerpo de una tabla
  • tfoot: Define el pie de una tabla

Atibutos para tr y td: combinación de celdas

En las tablas, podemos combinar celdas con el fin de poder establecer una estructura personalizada
con nuestro contenido.
Para ello, utilizaremos los atributos rowspan y colspan que aplicaremos en nuestras
celdas y filas, y que nos permitirá visualmente realizar combinaciones de celdas.
  • rowspan: indica el número de columna que ocupa cada celda, es decir, combinación de columnas.
  • colspan: indica el númoer de fila que ocupa cada celda, es decir, combinación de filas
