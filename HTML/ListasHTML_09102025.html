Listas en HTML5

En HTML5, disponemos de tres tipos de listas que nos van a permitir organizar texto para darle
un formato más conciso y claro.
Los tipos de listas que podemos utilizar, son los siguientes:
  • Listas Ordenadas (ol):Son listas ordenadas, y van numeradas en orden.
    La nomenclatura ol, vienen de Ordered List.
  • Listas DesOrdenadas (ul):Son listas desordenadas,es decir, no van en orden.
    La nomenclatura ul, viene de UnOrdered List.
  • Listas de Definición (dl): Son listas de definición.
    Las siglas dl, vienen dadas por Definition List.
Dentro de un elemento lista, sea del tipo que sea, podemos insertar cualquier tipo de información:
texto, imágenes, vídeos, tablas, audios, etc.
Por lo tanto, se pueden combinar listas ordenadas con desordenadas, y viceversa.

Listas Ordenadas (ol)

La estructura de una lista ordenada (ol) es la siguiente:
  • La lista está formada por la etiqueta de apertura y cierre ol
  • Internamente, dispondrá de una etiqueta li, en la que pondremos los elementos a listas.
    Es decir, el contenido de nuestra lista.
  • La etiqueta li, irá siempre tanto en listas ordenadas como desordenadas, porque son parte de su estructura.
Ejemplo de Lista Ordenada
  1. Seat Toledo
  2. Audi A3
  3. Toyota RAV4
  4. VW Golf
  5. Citroen Xsara

Listas Desordenadas (ul)

La estructura de una lista desordenada (ul) es la siguiente:
  • La lista está formada por la etiqueta de apertura y cierre ul
  • Internamente, dispondrá de una etiqueta li, en la que pondremos los elementos a listas.
    Es decir, el contenido de nuestra lista.
  • La etiqueta li, irá siempre tanto en listas ordenadas como desordenadas, porque son parte de su estructura.
Ejemplo de Lista Desordenada
  • Windows 11
  • Kali Linux
  • Ubuntu 24.04
  • Linux Mint
  • Red Hat Linux

Formateado de Listas Ordenadas y Desordenadas

HTML5, nos permite en cierta medida,poder customizar las listas a través de una serie de atributos
básicos como el atributo start y el atributo type.
Con estos atributos, podemos cambiar el orden del comienzo de las listas ordenadas,
y el tipo de icóno (es decir, podemos cambiar letas, números normales o romanos, etc.)
En el cado de las listas desordenadas, podemos poner icónos circulares (circle),
discos (disc) y cuadrados (square).

Ejemplo de Listas Ordenadas Aplicando Atributos type y start

  1. Podemos poner letras mayúscias (A)
  2. Podemos poner letas en mínuscula (a)
  3. Podemos poner cifras romanas en mayúsculas y minúsculas (i I, ii II,iii III, iv IV...n)
  4. Podemos poner números normales (1,2,3,4,5...n)
  5. Podemos indicar en qué posición queremos que comiencen. Utilizamos para ello el atributo start

Ejemplo de Listas Desordenadas Aplicando Atributos type

  • Podemos utilizar el tipo disc (viñeta redonda rellena)
  • Podemos utilizar el tipo circle (viñeta redonda hueca)
  • Podemos utilizar el tipo square (viñeta cuadrada)

Listas de Definición en HTML5 (dl)

Las listas de definición, tienen una estructura totalmente distinta a las anteriores vistas.
Tienen en este caso, la siguiente estructura:
  • La etiqueta dl: Description List. Delimitar la lista completa
  • La etiqueta dt: Description Term. Delimitar el término de la lista
  • La etiqueta dd: Definition Description. Delimita la definición del término

Ejemplo de Lista de Definición

dl
Crear la caja que contiene los items de la lista
dt
Es el item que contiene un término a definir.
dd
Es el item que contiene la definición.
