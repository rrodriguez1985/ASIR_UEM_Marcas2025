Definición de Página Web

  • Una pagina web, lo podemos definir como un documento de texto con marcas o etiquetas.
  • Estas etiquetas, lo que permiten, es presentar informaión al usuario: texto, imágenes, vídeos, fotos,tablas, etc.
  • Las marcas se llaman etiquetas o tags, y NO SON UN LENGUAJE DE PROGRAMACIÓN COMO JAVA, PYTHON, ETC.
  • Estas páginas estan pensadas para que sean visibles a través de los navegadores
  • Los navegadores web, se encargan por lo tanto de interpretar estas marcas con el fin de que los usuarios podamos visualizar y ver su contenido.

Definición de HTML

El acronimo HTML, viene del nombre HyperText Mackup Language, es decir, significa: Lenguaje de Marcado de Hipertexto.
Esto quiere decir, que las páginas web, son un conjunto de páginas con etiquetas que están vinculadas entre ellas a través de hiperenlaces o hipervínculos.
Las características que tenemos que tener en cuenta son las siguientes:
  • Que la estructura de HTML viene definida por la W3Consortium
  • Pueden aglutinar textos, audios, vídeos, tablas, imágenes,etc. que podemos combinar a nuestro gusto
    Además, permite insertar enlaces a otras páginas por medio de hiperenlaces.
  • HTML es un lenguaje de marcado o de marcas, y está diseñado para la visualización de información que se hace a través de los navegadores web.
  • Para que una página web sea considerada como tal, tiene que tener una extensión que sea .html. Ejemplo: index.html

Estructura o tags en HTML

Las etiquetas en HTML, se escriben de la siguiente manera:
  • Se escriben entre desigualdades, suelen ir en parejas y rodeando el texto al que aplica.
  • Tienen una etiqueta de principio y una de fin. Es decir, etiqueta de apertura y etiqueta de cierre
  • La parte de apertura contiene como mínimo el nombre de la marca o etiqueta, y el de cierre, empieza con el caracter la / seguido del nombre de la etiqueta

En HTML podemos encontrarnos con etiquetas especiales que sólo tienen etiqueta de apertura.
Es lo que se conoce como monoetiquetas. Eje: img, hr, br, etc.
Las etiquetas que ya tengan etiqueta de apertura y de cierre o que sean monoetiquetas, pueden tener atributos.
Los atributos son cualidades que facilitan a las etiquetas el poder dotarles de una serie de características concretas.

Estructura de una página web y comentarios

Comentarios en HTML

Los comentarios en HTML, o en CSS, se utilizan para dar información al progradxr/a con el fin de informar de las cosas que vamos haciendo.
Los comentarios deben ser claros, sencillos y conciso.
Los comentarios NO van a aparecer en el navegador, sino que quedan de manera intenrta para poder ser visualizados por los programadores.
Para poder poner los comentarios en HTML, utilizaremos la siguiente sintaxis:
  • Comenzaremos con el símbolo de menr que '<' seguido de '!' y de '--'
  • Introducimos el texto con el mensaje a comentar
  • Cerramos con la estructura '--' seguido de '>'

Estructura de un documento HTML

  • Los docuemntos HTML, están divididos en dos grandes zonas principales: head y el body
  • head:El head o cabecera de una página web, se caracteríza porque es la zona en donde almacenaremos
    el nombre de la pestaña (title), favicon, librerías externas de CSS y JS, metadatos, idioma, etc.
  • body: El body o cuerpo del documento HTML, es donde vamos a introducir TODA la información de nuestra web: imágenes, texto, vídeos, audios, formularios, etc.

Etiquetas que ya no se utilizan en HTML5

Con la llegada de HTML5, algunas etiquetas que hasta entonces utilizábamos para poder dar apariencia y formate en nuestras webs, ahora quedan en deshuso.
A partir de ahora, todo aquello que vaya relacionadocon apariencia (color, tipografías, alineación, tamaños, etc.), lo haremos con la etiquetas y reglas establecidas en CSS.
Para poder cambiar tamaños, tipografías y color en HTML, utilizaremos la etiqueta font
A parte de la etiqueta de apertura y cierre font, utilizaremos los siguientes atributos:
  • face: sirve para cambiar el tipo de texto. Ejemplo: Arial, TimeNewRoman, Verdana, etc.
  • color: sirve para establecer un color en el texto
  • size: sirve para dar un tamaño al texto. Siempre en px (Pixels)
