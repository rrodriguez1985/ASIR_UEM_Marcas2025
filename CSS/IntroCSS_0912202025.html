Introducción a CSS

Cuando hablamos de CSS, nos estamos refiriendo a Cascading Style Sheet.
Estas siglas, significan exactamente "Hojas de Estilo en Cascada"
En las primeras versiones de HTML, el código fuente de una página web tenía
tanto la información del contenido como de la forma / apariencia en la que se
representa.
Hoy en día, estos dos aspectos van separados. Es decir:
  • Por un lado tenemos el contenido: imágenes, tablas, vídeos, texto, etc.
  • Por otro lado, indicamos cómo va a ser ese contenido: tamaño de texto,
    tamaño de imágenes, alienaciones, colores, tipografías, etc.

En conclusión, por un lado tendremos nuestro archivo HTML, y por otro lado, tendremos
el archivo CSS o la apariencia que le queremos aplicar a nuestra página web.

En conclusión: HTML define el contenido y estructura de nuestra web.
Y CSS, define cómo será nuestra web, es decir, la apariencia.

Funcionamiento de CSS

El funcionamiento de las hojas de estilo es el siguiente:
  • En las páginas web (archivos .html) se escriben etiquetas que definen
    las categorías o elementos del contenido.
  • En las hojas de estilo (archivos .css) se escribe cómo queremos que sea
    el estilo de presentación de estas etiqeutas o categorías: fuentes,
    bordes de imágenes y/o tablas, posición en la web (centrada, izq. der, etc)...
  • En la página web se escribe qué hoja de estilo queremos utilizar.

En conclusión, podemos resumir el funcionamiento de CSS en los siguientes puntos clave:
  • Las hojas de estilo en cascada (CSS), son complementos a las etiquetas
    HTML, y lo que permiten es mejorar su presentación y aspecto.
  • Los CSS separan el contenido del documento de todo el aspecto de la presentación de este.
  • Intentar en la medida de lo posible, utilizar muchas tablas en las páginas web.
    Hoy en día, se utilizan más div o etiquetas semánticas predefindidas que tablas
    a la hora de hacer páginas web responsive (adaptablas a distintos formatos de pantallas)
  • No se indica en el código HTML el estilo del documento, sino que se indica lo que es cada
    elemento y dicho formato se especifica en el CSS
  • Todas estas especificaciones están recogidas en HTML5 y se utilizarán a partir de ahora como
    estandarización a la hora de realizar nuestras páginas web.

Versiones de CSS y cosas a tener en cuenta

Lo que tenemos que tener en cuenta a la hora de crear nuestras páginas web y sobre todo de usar CSS,
es que NO TODOS LOS NAVEGADORES WEB SOPORTAN TODAS LAS CARACTERÍSTICAS DE CSS DISPONIBLES
CSS dispone de tres versiones como principales o las más características:
  • CSS 1: Creada en 1996 - Propiedades fuentes, colores, alineaciones, etc.
  • CSS2 y CSS2.1: Creada en 1998 y 2005 respectivamente. Propiedades de posicionamiento así como
    mejoras en la aplicación en ciertas etiquetas de HTML.
  • CSS3: Creada desde el año 2011 hasta la actualidad*. Inserta funcionalidades varias que permiten
    utilizar CSS como un elemento fundamental para hacer páginas web dinámicas y más atractivas
    para los distintos dispositivos móviles existentes.
    *:Hoy en día, es más común utilizar CSS con las propiedas de animaciones, transiciones, transformaciones, etc.
    para hacer páginas web dinámicas, que utilizar lenguajes com JS (JavaScript) para realizar este tipo de funcionalidades.
    Desde hace unos años, ha surgido la idea de poder utilizar funciones y variables que se utilizan por lo general
    en lenguajes de programación, en CSS. Este tipo de acciones es lo que ha dado origen a la tecnología SASS.

Ventajas y Desventajas de CSS

Ventajas de CSS

  • CSS nos permite un mayor control de la presentación del sitio web
  • Si necesitamos hacer modificaciones ede presentación en un documento web, al utilizar CSS
    sólo lo haríamos en un único documento/apartado y no tendríamos que ir modificando
    el documento HTML parte por parte.
  • Mayor Legilibilidad --> más fácil de interpretar y entender
  • Se reduce duplicidad de estilos en diferentes lugares: ahora tenemos todo más organizado
    y colocado que antes. Es mucho más fácil modificar y cambiar algo que cómo lo teníamos
    organizado anteriormente.
  • Es más fácil crear distintas versiones de una página web y css para distintos dispositivos móviles

Desventajas de CSS

  • A veces, dependiendo del navegador web que utilicemos, la web se verá
    de una manera u otra...NO todos los navegadores soportan todas las etiquetas
    CSS disponibles.
  • El uso de las tablas nos permitían hasta ahora crear diseño complejos de forma mucho
    más sencilla que si utilizáramos CSS... Ahora, para poder "centrar" una
    caja o texto, tenemos que "jugar" con los distintos elementos de CSS

Cómo usar CSS en nuestras páginas web

Existen tres formas distintas de poder utilizar CSS en nuestras páginas web:
  • CSS Externo: Linkeamos en la cabecera de nuestra página web (head),
    la ruta del documento .css que queremos aplicar a nuestra página.
    La estructuctura es la siguiente:
    link rel=stylesheet type=text/css href="ruta_archivo.css"
  • CSS Interno:Dentro de la etiqueta head, se establecen etiquetas de apertura
    y cierre style, y dentro de estas, pondremos las características que queremos
    cambiar.
  • CSS en Línea o Embebido: es el que se aplica directemente en la etiqueta que queremos modificar.
    Para ello, aplicaremos la etiqueta style dentro de la etiqueta HTML que queremos modificar,
    y le daremos las propiedades correspondientes.

  • Para hacer comentarios de una sola línea o de varias líneas: /*......*/
